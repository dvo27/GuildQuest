All functionalities are displayed through the Tkinter GUI. To run this implementation of GuildQuest, please run main.py.
To begin, please click the register button to create a new user profile in GuildQuest. Then, you can begin creating a campaign,
quest events, and characters. Default realms are already provided and there to choose from for your campaigns and quest events.
This implementation is constructed with Python 3.11. No extra dependencies should be needed.
